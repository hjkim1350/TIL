#### 6044번 [기초-산술연산] 정수 2개 입력받아 자동 계산하기

```python
# 입력받은 공백으로 구분된 2개의 정수(a, b) 합, 차, 곱, 몫, 나머지, 나눈 값 계산
# 단 b는 0이 아님을 전제

a, b = input().split(' ')
a = int(a)
b = int(b)
print(a + b)
print(a - b)
print(a * b)
print(a // b)
print(a % b)
print(format(a / b,".2f"))
```
